# 2026 Q1 产品经营离线知识库 Mock 文档

这组文档用于支撑当前季度工作汇报 demo 中的离线知识库场景，内容与页面里的知识库选择和中间检索过程保持一致。

## 文档清单

1. `25年产品季报.docx`
   用途：支持“结构与章节规划检索”
2. `26年产品月报.docx`
   用途：支持“结构与章节规划检索”以及月报向季报的汇总表达
3. `常见经营分析案例.docx`
   用途：支持“经营分析与口径补齐”
4. `经营问题经典case.docx`
   用途：支持“风险表达与案例校验”
5. `产品经营数据汇报数据口径.docx`
   用途：支持“经营分析与口径补齐”以及指标口径统一

## 生成方式

运行下面的脚本可重新生成全部 Word 文档：

```bash
python3 scripts/generate_quarterly_report_kb_docs.py
```

## 输出目录

生成后的 `.docx` 文件会落在当前目录：

`mock-data/offline-knowledge-base/2026-q1-product-ops/`
